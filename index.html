Installation

1

Open Chrome Extensions

Navigate to chrome://extensions/

2

Enable Developer Mode

Toggle the switch in the top-right corner

3

Load Unpacked

Click Load unpacked and select the agave folder

4

You're Done!

Extension is now installed and ready to use

How to Use

  1. Visit any supported shopping site (nordvpn.com, amazon.com, skillshare.com, audible.com, squarespace.com)
  2. A floating green widget appears in the top-right corner
  3. Browse the list of creators or use the search box to filter
  4. Click any creator to apply their affiliate link
  5. The page reloads with the creator's affiliate URL for "last-click" attribution
